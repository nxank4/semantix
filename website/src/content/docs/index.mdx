## Quick Example

```python
from pydantic import BaseModel
import loclean

class Product(BaseModel):
    name: str
    price: int
    color: str

# Extract from text
item = loclean.extract("Selling red t-shirt for 50k", schema=Product)
print(item.name)   # "t-shirt"
print(item.price)  # 50000
print(item.color)  # "red"
```
